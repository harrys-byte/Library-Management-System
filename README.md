# Library-Management-System
A web-based system built using the famous stack (Frontend: HTML, CSS | Backend: JavaScript) for book management and purchasing, with admin-controlled operations. Utilized MS Excel as a lightweight database.

# Work Flow

- Create an admin
- Add the book details and prices
- Download the list of books as .xlsx using the download option
- Create users for the customers
- Search/filter books
- Add to cart
- Proceed to checkout
